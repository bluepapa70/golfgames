# 📋 프로젝트 계획서: 대한민국 아마추어 골프 게임 도감 (GolfGameCard)

본 프로젝트는 대한민국 아마추어 골퍼들이 즐기는 10가지 골프 게임 정보를 한눈에 보고, 각 게임 카드를 클릭하면 뒤집히는(Flip) 애니메이션과 함께 상세 룰을 확인할 수 있는 반응형 웹사이트 개발 계획입니다.

---

## 1. 프로젝트 개요
* **목적**: 필드나 스크린 골프 대기 중 동반자들이 스마트폰으로 쉽고 빠르게 게임 룰을 찾아볼 수 있는 가이드 제공
* **핵심 기능**:
  * 10개 골프 게임의 인기도, 난이도, 요약이 담긴 카드 그리드 배치
  * 카드를 클릭(또는 터치)하면 CSS Flip 애니메이션 적용 (앞면: 요약 / 뒷면: 상세 플레이 방법)
  * 인기도/난이도 정렬 및 필터링 기능
* **타깃 디바이스**: 모바일 웹(Mobile-First), 태블릿, PC 반응형

---

## 2. 기술 스택 (Tech Stack)
* **Frontend**: Vanilla HTML + CSS + JavaScript (빌드 도구 없음, 파일 직접 열기 가능)
* **Styling**: 순수 CSS (Flexbox + Grid + CSS Variables), `golfgame.css` 기반 확장
* **Animation**: CSS 3D Transforms (`perspective`, `rotateY`) — `golfgame.css` 스켈레톤 활용
* **Deployment**: GitHub Pages (정적 파일 직접 배포)
* **Data**: `games-data.js` — JS 모듈로 게임 데이터 관리

## 5. 데이터 구조 (Data Schema)

`games-data.js`에 아래 구조로 10개 게임을 정의합니다.

```js
// games-data.js 상단 예시
export const GAMES = [
  {
    id: 1,
    name: "라스베가스",
    nameEn: "Las Vegas",
    emoji: "🎰",
    popularity: 5,      // 1~5 (별점)
    difficulty: 2,      // 1~5 (별점)
    summary: "매 홀 팀이 바뀌는 스릴 넘치는 2:2 팀 게임",
    target: "초보·고수가 섞인 4인 조",
    tags: ["팀게임", "초보추천"],
    description: [...],
    tip: "..."          // 고수 팁 (없으면 null)
  }
]
```

| 필드 | 타입 | 설명 |
|------|------|------|
| `id` | number | 고유 ID |
| `name` | string | 한글 게임명 |
| `nameEn` | string | 영문 게임명 |
| `emoji` | string | 대표 이모지 |
| `popularity` | 1~5 | 인기도 |
| `difficulty` | 1~5 | 난이도 |
| `summary` | string | 앞면 한 줄 요약 |
| `target` | string | 추천 대상 |
| `tags` | string[] | `["초보추천"]` `["고수용"]` `["팀게임"]` 등 |
| `description` | string[] | 뒷면 룰 설명 배열 |
| `tip` | string \| null | 고수 팁 (선택) |

## 6. 10개 골프 게임 데이터

### 1. 라스베가스 (Las Vegas) 🎰
* **인기도**: ★★★★★ (5) / **난이도**: ★★☆☆☆ (2)
* **요약**: 매 홀 편이 바뀌는 최고의 우정 유지 게임
* **대상**: 초보·고수가 섞인 4인 조
* **태그**: 팀게임, 초보추천
* **룰**:
  1. 전 홀 스코어 기준 1등+4등 vs 2등+3등으로 팀 구성 (첫 홀은 랜덤)
  2. 각 팀원 2명의 타수를 합산, 낮은 팀이 승리
  3. 타수 차이 × 미리 정한 단가만큼 이긴 팀이 받음
* **고수 팁**: 합산 대신 이어 붙이기(4타+5타=45점) 방식으로 극적인 역전 가능

---

### 2. 뽑기 (조커 게임) 🎴
* **인기도**: ★★★★★ (5) / **난이도**: ★★☆☆☆ (2)
* **요약**: 뽑기통 하나로 실력 불문 평등한 복불복
* **대상**: 100타 이상 초보자 포함 조
* **태그**: 팀게임, 초보추천
* **룰**:
  1. 매 홀 티잉 그라운드에서 4명이 뽑기통(번호 막대 또는 카드)을 뽑아 팀을 결정 (예: 1·3번 vs 2·4번)
  2. 팀별 타수 합산 또는 낮은 타수(베스트볼)로 홀 승패를 가리고 단가 정산
  3. 조커 카드를 뽑은 사람은 해당 홀에서 1인 팀이 되어 나머지 셋을 상대하거나 특별 혜택(선택 합의)을 받음
* **고수 팁**: 뽑기 대신 동전 던지기·사다리타기로 변형 가능 — 실력보다 운이 지배해 초보도 즐거운 룰

---

### 3. 스킨스 (Skins) 💰
* **인기도**: ★★★★☆ (4) / **난이도**: ★★★☆☆ (3)
* **요약**: 홀별 각자도생, 이월되는 상금의 짜릿함
* **대상**: 실력이 엇비슷한 동반자
* **태그**: 개인게임, 중수추천
* **룰**:
  1. 매 홀마다 일정 금액(스킨)을 걸고, 단독 최저타를 기록한 사람이 해당 홀 스킨 획득
  2. 공동 최저타 시 스킨이 다음 홀로 이월 (누적)
  3. 18홀 후 획득한 스킨 수 × 단가 정산
* **고수 팁**: 이월이 쌓인 홀일수록 압박감 극대화 — 버디 이하를 노리는 공격적 플레이 유도

---

### 4. OECD 스킨스 🌍
* **인기도**: ★★★★☆ (4) / **난이도**: ★★★★☆ (4)
* **요약**: 돈을 따면 덫에 걸리는 한국형 견제 룰
* **대상**: 보기 플레이어 이상의 중급자
* **태그**: 개인게임, 중수추천
* **룰**:
  1. 기본 스킨스와 동일하게 진행하되, 스킨을 획득한 사람이 다음 홀에서 자동으로 1인 팀(OECD 선진국)이 됨
  2. OECD 역할이 된 사람은 해당 홀에서 이기면 2배 획득, 지면 나머지 3명에게 각각 단가 지급
  3. 연속 스킨 획득 시 매번 OECD가 되어 부담이 배가 — 잘 치면 치를수록 위험해지는 구조
* **고수 팁**: 이긴 홀 직후 일부러 힘 빼는 전략이 통하는 유일한 게임 — 심리전이 핵심

---

### 5. 스트로크 (Stroke) 📏
* **인기도**: ★★★☆☆ (3) / **난이도**: ★★★★★ (5)
* **요약**: 1타당 금액 책정, 냉정한 실력 위주의 정통 방식
* **대상**: 싱글 골퍼 및 실력파 타짜들
* **태그**: 개인게임, 고수용
* **룰**:
  1. 라운드 시작 전 타당 단가를 합의 (예: 1타당 1,000원)
  2. 18홀 종료 후 4명의 총 타수를 비교, 가장 적게 친 사람 기준으로 타수 차이만큼 정산
  3. 핸디캡 적용 여부와 상한선(캡)을 반드시 사전에 합의
* **고수 팁**: 더블보기 이상 홀이 하나라도 나오면 순식간에 손해가 눈덩이처럼 불어나는 냉혹한 게임

---

### 6. 후반 조인 (하이로우) ⚖️
* **인기도**: ★★★★☆ (4) / **난이도**: ★★☆☆☆ (2)
* **요약**: 전반 스코어로 편을 짜서 후반에 승부
* **대상**: 전반전 끝나고 분위기 반전용
* **태그**: 팀게임, 초보추천
* **룰**:
  1. 전반 9홀이 끝난 후 스코어를 나열 (예: A 40타, B 45타, C 48타, D 52타)
  2. 1등(A)과 4등(D)이 한 편, 2등(B)과 3등(C)이 한 편이 되어 후반 9홀 팀전 진행
  3. 매 홀 각 팀의 낮은 타수(Low)끼리 비교하거나 합산 타수로 홀별 승패를 가름
* **고수 팁**: 실력 밸런스가 가장 잘 맞는 정교한 팀 게임 — 전반에 잘 쳐도 방심하면 후반에 독박

---

### 7. 좌탄우탄 (티샷 게임) 🏌️
* **인기도**: ★★★★☆ (4) / **난이도**: ★☆☆☆☆ (1)
* **요약**: 티샷 구질과 방향으로 즉석 팀 결성
* **대상**: 드라이버 난조로 고생하는 조
* **태그**: 팀게임, 초보추천
* **룰**:
  1. 4명이 차례대로 티샷을 마친 후, 페어웨이 기준으로 공이 떨어진 위치를 확인
  2. 가장 왼쪽으로 간 2명(좌탄)이 한 편, 오른쪽으로 간 2명(우탄)이 한 편이 됨
  3. 3:1로 쏠릴 경우 가장 멀리 친 사람과 홀로 남은 사람이 한 편이 되는 로컬 룰 적용
* **고수 팁**: 편이 매번 황당하게 짜여 카트 안이 웃음바다 — 드라이버 OB도 예측 불가 팀 빌딩의 재료

---

### 8. 방울뱀 (Snake) 🐍
* **인기도**: ★★★☆☆ (3) / **난이도**: ★★★★☆ (4)
* **요약**: 쓰리퍼트(3-Putts) 독박 배설 게임
* **대상**: 그린 위 숏게임 긴장감 강화용
* **태그**: 개인게임, 고수용
* **룰**:
  1. 누군가 그린 위에서 쓰리퍼트를 하는 순간 방울뱀 마커(동전·인형 등)를 인수
  2. 마커 소지자는 다음 사람이 쓰리퍼트를 하기 전까지 매 홀 벌금이 누적됨
  3. 18번 홀 마지막 그린에서 마커를 들고 있는 사람이 누적 벌금 전액 독박 납부
* **고수 팁**: 마지막 홀 그린 주변의 긴장감이 극대화 — 변형으로 포퍼트(4퍼트)만 적용해 진입 장벽 낮춤

---

### 9. 딩동댕 (Ding Dong Dang) 🔔
* **인기도**: ★★★☆☆ (3) / **난이도**: ★★☆☆☆ (2)
* **요약**: 홀컵과의 거리에 따라 단계별 점수 획득
* **대상**: 비거리는 짧으나 정확도 높은 조 / 시니어·여성 골퍼 포함 조
* **태그**: 개인게임, 초보추천
* **룰**:
  1. **딩**: 티샷을 날려 페어웨이에 가장 멀리(또는 가장 먼저) 안착시킨 사람 → 1점
  2. **동**: 그린에 가장 먼저 공을 올린(On-Green) 사람 → 1점
  3. **댕**: 그린 위에서 가장 먼저 홀컵에 공을 집어넣은(Hole-Out) 사람 → 1점
* **고수 팁**: 고수가 멀리 쳐도 세컨샷 실수로 그린을 못 잡으면 하수가 동·댕을 가져갈 수 있어 모두에게 기회 공평

---

### 10. 포섬 / 포볼 (Match Play) 🤝
* **인기도**: ★★★☆☆ (3) / **난이도**: ★★★☆☆ (3)
* **요약**: 정통 팀 대항 매치플레이 방식
* **대상**: 동호회 대항전 및 커플·부부 동반 라운딩
* **태그**: 팀게임, 중수추천
* **룰 (포볼)**:
  1. 2인 1조, 총 4명이 각자의 공으로 플레이
  2. 홀이 끝난 후 우리 팀 2명 중 더 좋은 점수를 팀 성적으로 채택하여 상대 팀과 비교
  3. 홀 승패를 누적해 더 많은 홀을 이긴 팀이 최종 승리 (부담이 적어 인기 많음)
* **룰 (포섬)**:
  1. 2인 1조가 하나의 공을 번갈아 가며 침 (A 티샷 → B 세컨샷 → A 어프로치 → B 퍼팅)
  2. 파트너 간 호흡이 절대적 — 실수를 서로 받아줘야 하므로 난이도 매우 높음
* **고수 팁**: 포볼은 동호회 신입 환영 라운딩에 적합, 포섬은 부부 동반 라운딩에서 화합과 갈등을 동시에 유발

## 7. 구현 단계 (Implementation Steps)

```
Step 1. 데이터 파일 작성
  → games-data.js 생성 후 콘솔에서 데이터 로드 확인

Step 2. HTML 뼈대 작성
  → index.html: Header / Filter Bar / Card Grid / Footer 구조 완성
  → 스크린리더 접근성 (aria-label, role) 포함

Step 3. CSS 완성
  → golfgame.css 확장: 레이아웃 변수, 그리드, 카드 스타일, Flip 완성
  → 모바일 → 태블릿 → PC 반응형 미디어 쿼리

Step 4. JS 렌더링 구현
  → app.js: 카드 동적 생성 + 그리드 삽입

Step 5. Flip 이벤트 연결
  → 카드 클릭 시 is-flipped 클래스 토글

Step 6. 필터 & 정렬 구현
  → 버튼 클릭 시 games 배열 필터 후 리렌더

Step 7. 최종 검증
  → 모바일(Chrome DevTools) / 태블릿 / PC 레이아웃 확인
  → 전 카드 Flip 동작 확인
  → 필터/정렬 전 카드 전환 확인
```
